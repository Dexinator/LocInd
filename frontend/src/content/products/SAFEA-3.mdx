## Descripción Ampliada
El modelo Safe A-3 es la base para el sistema de localización en tiempo real. Funciona como detector de otros dispositivos y puede agregar la información para luego entregarla a otros sistemas de monitoreo. Se pueden concatenar anclas para crear un sistema de geocercado (geofencing) en uno o varios sitios. El sistema es de alta precisión, robusto y resistente a interferencia del medio.
